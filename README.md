# Sample JS Project

This is a simple HTML + JavaScript project created for learning **Git** and **GitHub**.

## Files
- `index.html` → Main HTML page
- `script.js` → JavaScript with some utility functions

## How to Run
1. Open `index.html` in your browser.
2. Click the button to trigger JavaScript.
3. Open the browser console to see logs.

## Git Commands to Push
```bash
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/sample-js-project.git
git push -u origin main
